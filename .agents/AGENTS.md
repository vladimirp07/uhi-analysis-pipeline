# Reglas de Proyecto (Project Rules)

Este archivo contiene reglas y restricciones de comportamiento específicas para este espacio de trabajo.

## Git y Control de Versiones
*   **No hacer Push Automático**: Al realizar cambios en el repositorio, prepara los archivos (`git add`) y realiza la confirmación local (`git commit`), pero **nunca ejecutes `git push` directamente**. Deja que el usuario revise y envíe los cambios de forma manual.
